#项目描述
	记录cloa生活中的点滴
	angular 1.7.2 + bootstrap 3


--index.html                          主文件
--main.js                             入口文件：引起一些依赖文件和启动angular
--readme.md 						  项目简介
--lib								  存放项目引用的框架文件
--src                                 项目资源
  |--controllers					  控制器
  |--directives	                      自定义指定
  |--fliters                          自定义过滤器
  |--services                         自定义服务
  |--route                            路由文件
  |--views                            视图html
--change.log                          日志文件：记录每次重要功能合入
